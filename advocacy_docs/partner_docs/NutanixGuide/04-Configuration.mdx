## Deploying VMs Using AHV

The following steps outlined below describe how to create a Virtual Machine (VM) via AHV
and install the following EDB products on it:

- **EDB Postgres Advanced (EPAS)**
- **EDB Failover Manager (EFM)**
- **Postgres Enterprise Manager (PEM)**
- **Backup Recovery Manager (Barman)**

1. **Create a VM**

On the Prism Central web console, select **Create VM**. Enter the desired specs for your VM and click **Save**.

<p align="center">
  <img  src="Images/CreateaVM.png">
</p>

For additional information on how to create a VM on AHV you can go to this [link](https://www.youtube.com/watch?v=q4wBewXfDs8) from Nutanix.

**Below are the specifications used for creating the test environment.**

Use the appropriate values for your configuration.

    vCPU(s): 2
    Memory: 4 GiB
    Storage: DISK 50 GiB
    Guest OS: CentOS7

!!! Note
    Mount the CD-ROM with CentOS7 ISO available via the Image Service

<p align="center">
  <img  src="Images/TestEnvironmentSpecifications.png">
</p>

2. **Install EDB Postgres Advanced (EPAS).**

Refer to the EDB documentation for installing EPAS (versions 10 through 13):

https://www.enterprisedb.com/docs/epas/latest/epas_qs_linux_7/

https://www.enterprisedb.com/docs/epas/12/

https://www.enterprisedb.com/docs/epas/11/

https://www.enterprisedb.com/docs/epas/10/


3. **Install all other EDB Tools needed for your configuration in the appropriate Virtual Machines.**

Refer to the [EDB documentation](https://www.enterprisedb.com/edb-docs) for installing EDB Tools.
