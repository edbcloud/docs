The following section describes numeric bitwise functions that accepts numeric input and returns numeric bitwise values.

## BITAND

The `BITAND` function performs a bitwise `AND` operation and returns an numeric value based on the data type of input argument.

**Synopsis**

```text
BITAND(expr1, expr2)
```

The input data type of `expr1` and `expr2` specifies an `NUMERIC` value on which the bitwise `AND` operation is performed. If either argument to `BITAND` is `NULL`, the result is a `NULL` value.

The `BITAND` compares each bit of `expr1` value to the corresponding bit of `exp2` using a bitwise `AND` operation and converts the value of a input argument to `NUMERIC`.

**Return Types**

The return type is determined by the input data type of `expression`. The following table illustrates the return type for each input type.

| Input Type  | Return Type |
| ----------- | ----------- |
| `NUMERIC`   | `NUMERIC`   |
| `BIGINT`    | `BIGINT`    |
| `INTEGER`   | `INTEGER`   |
| `SMALLINT`  | `SMALLINT`  |
| `INET`      | `INET`      |
| `MACADDR`   | `MACADDR`   |
| `MACADDR8`  | `MACADDR8`  |

**Examples**

The following example performs an `AND` operation on the numbers 10 and 11.

```text
edb=# SELECT BITAND(10,11) FROM DUAL;
 bitand 
--------
     10
(1 row)
```

# BITOR

The `BITOR` function performs a bitwise `OR` operation and returns an numeric value based on the data type of input argument.

**Synopsis**

```text
BITOR(expr1, expr2)
```

The `BITOR` compares each bit of `expr1` value to the corresponding bit of `exp2` using a bitwise `OR` operation and converts the value of a input argument to `NUMERIC`.

**Return Types**

The return type is determined by the input data type of `expression`. The following table illustrates the return type for each input type.

| Input Type  | Return Type |
| ----------- | ----------- |
| `NUMERIC`   | `NUMERIC`   |
| `BIGINT`    | `BIGINT`    |
| `INTEGER`   | `INTEGER`   |
| `SMALLINT`  | `SMALLINT`  |
| `INET`      | `INET`      |
| `MACADDR`   | `MACADDR`   |
| `MACADDR8`  | `MACADDR8`  |

**Examples**

The following example performs an `OR` operation on the numbers 10 and 11.

```text
edb=# SELECT BITOR(10,11) FROM DUAL;
 bitor 
-------
    11
(1 row)
```

# BITWISE OPERATORS

The following table describes the bitwise operators, which performs the same operation as `BITAND` and `BITOR`.

| Operator | Description                                                                                      |
| -------- | ------------------------------------------------------------------------------------------------ |
| `&`      | `(Bitwise AND)` Returns the result of a bitwise `AND` operation performed on two numeric values. |
| `|`      | `(Bitwise OR)` Returns the result of a bitwise `OR` operation performed on two numeric values.   |

**Examples**

The following example uses `&` operator and returns the output as `BITAND` of the input.

```text
edb=# SELECT 10&11 FROM DUAL;
 ?column? 
----------
       10
(1 row)
```

The following example uses `|` operator and returns the output as `BITOR` of the input.

```text
edb=# SELECT 10|11 FROM DUAL;
 ?column? 
----------
       11
(1 row)
```
